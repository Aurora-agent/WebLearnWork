# 🌱 我的成长笔记

> 记录学习点滴，分享生活感悟

基于 HTML + CSS 构建的个人博客网站，四川大学计算机学院《网页设计》课程期末大作业。

---

## 📖 页面导航

| 页面 | 说明 |
|------|------|
| **首页** | 博客主入口，展示核心导航卡片 |
| **关于我** | 个人简介、教育背景与技术技能 |
| **兴趣爱好** | 排球运动与二次元世界的热情分享 |
| **我的大学** | 江安校区与望江校区的校园风光 |
| **节日寄语** | 端午节、春节、中秋节传统节日特辑 |
| **动漫推荐** | 银魂 & 紫罗兰永恒花园 推荐 |
| **排球队伍** | 日本国家排球队介绍 |

---

## 🛠 技术栈

- **HTML5** — 语义化标签，Dreamweaver 模板驱动
- **CSS3** — 响应式布局、网格系统、CSS 变量与动画
- **纯静态** — 无需构建工具，开箱即用

---

## 🗂 项目结构

```
myweb/
├── index.html            # 首页
├── introduction.html     # 关于我
├── hobby.html            # 兴趣爱好
├── university.html       # 江安校区
├── wangjiang.html        # 望江校区
├── festival.html         # 端午节日
├── spring.html           # 春节特辑
├── mid-autumn.html       # 中秋特辑
├── anime.html            # 动漫推荐
├── volleyball.html       # 排球队伍
├── css/
│   ├── style.css         # 全局样式
│   └── *.css             # 各页面独立样式
├── img/                  # 图片资源
└── Templates/            # Dreamweaver 模板
```

---

## 🚀 使用方式

直接浏览器打开 `index.html` 即可浏览全站，各页面通过导航栏相互链接。
